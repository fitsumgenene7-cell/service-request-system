Service Request System

A desktop application built with Java and JavaFX to manage service requests efficiently. Users can create, track, and update requests, while admins can manage and monitor all requests in the system.

⸻

Features
	•	Create and submit service requests with details like title, description, priority, and department.
	•	Track request status: Pending, In Progress, Completed.
	•	Admin panel for managing, assigning, and updating requests.
	•	Search and filter requests based on status, priority, or assignee.
	•	User role management with different access permissions.

⸻

Technologies Used
	•	Java 17+
	•	JavaFX 20+ for GUI
	•	SQLite (or any relational database)
	•	Maven for dependency management
